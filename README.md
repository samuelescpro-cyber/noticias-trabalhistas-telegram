# Bot Telegram - Notícias (Justiça do Trabalho / TRT23)

Este projeto coleta links de fontes definidas e envia para o Telegram apenas notícias que:
1) Batem nos termos (processo/decisão/recursos etc. — sua lista `KEY_PHRASES`)
2) Têm sinais de Justiça do Trabalho (`JT_STRONG` / `JT_WEAK`, com `STRICT_JT=1`)
